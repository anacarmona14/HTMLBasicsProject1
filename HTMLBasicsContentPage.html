Early 90s - The first website ever published and the beginning of the web as we know it

Tim Berners-Lee and his team launched the world's first web page on August 6, 1991. This modest page, hosted on a NeXT computer at CERN, contained information about the World Wide Web and how to create web sites. There were no graphic components on these text-only websites. The first web browser, also created by Tim Berners-Lee, was called WorldWideWeb (later renamed Nexus). It was a graphical browser launched on 1990 that ran on the NeXTSTEP operating system and allowed users to create and edit web pages as well as browse them.

Another early browser was the Line Mode Browser, created by Nicola Pellow in 1991. It was a text-based browser that allowed users to browse the web without requiring a graphical interface.

Other early web browsers included Mosaic (see screenshot below), which was released in 1993 and was the first browser to display images inline with text, and Netscape Navigator, which was released in 1994 and became the
dominant browser of the mid-1990s.

Mid 90s and early 2000s - Graphic browsers and the era of JavaScript + CSS + Flash

As previously mentioned, graphic browsers, such as Netscape Navigator, Mosaic and the infamous Internet Explorer, began to overtake text-based browsers, allowing graphics and more visually pleasing webpages to emerge. This enabled designers to develop more sophisticated websites, and consumers to interact with web material in a more natural manner. JavaScript and CSS were introduced as the perfect complement for HTML websites, along with the Flash elements that characterized the 2000s. JavaScript enabled the development of interactive web content such as pop-up windows, drop-down menus, and other features. CSS, on the other hand, gave designers more control over how web information was displayed, making it easier to construct more visually pleasing websites with uniform layouts.
